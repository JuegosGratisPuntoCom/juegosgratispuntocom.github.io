Juegos Gratis Punto Com (Free Games Dot Com) is an unblocked free browser games site

Some networks (like schools, businesses or libraries) may block your regular game sites,
but likely they are not blocking GitHub Pages, so you can still access JuegosGratisPuntoCom from them.

JGPC started as a fork of games-site.github.io made by PepeBigotes,
all credit for the basic website structure and adding most games goes to the original creators of games-site.

Compared to the original site, this fork changes the design, look, functionality and adds more games regularly!
